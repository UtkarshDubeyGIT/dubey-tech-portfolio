## Overview

This research project addresses the critical trade-off between **predictive accuracy** and **regulatory interpretability** in financial credit risk modeling. We propose a novel **Fuzzy-Monotonic LightGBM** framework that achieves competitive performance (ROC-AUC ~0.77, PR-AUC 0.55) while maintaining structural transparency through:

- üîß **Behavioral Feature Engineering**: Domain-driven credit indicators
- üîç **Multi-Layer Explainability**: Structural (fuzzy + monotonic) + attributional (SHAP)

## Architecture

The framework consists of multiple interconnected layers:

1. **Data Preprocessing Layer**: Feature alignment, normalization (RobustScaler), and categorical encoding
2. **Behavioral Feature Engineering**: Domain-driven credit indicators including utilization rates, delinquency intensity, and payment trends
3. **Fuzzy Membership Layer**: Linguistic variables (Low/Medium/High) with percentile-based cut-points
4. **Monotonic LightGBM Ensemble**: Gradient boosted decision trees with economic constraint enforcement
5. **Explainability Layer**: SHAP feature attribution combined with fuzzy rule activations

## Methodology

### Behavioral Feature Engineering

```python
# Aggregated spending behavior
BILL_AMT_AVG = mean(BILL_AMT1, ..., BILL_AMT6)

# Credit utilization
utilization = BILL_AMT_AVG / LIMIT_BAL  # clipped [0, 1]

# Repayment discipline
repay_ratio1 = PAY_AMT1 / BILL_AMT1  # clipped [0, 1]

# Delinquency severity
delinquency_intensity = sum(PAY_0, ..., PAY_6)

# Payment trend (behavioral drift)
paytrend = PAY_6 - PAY_0  # improving ‚Üí negative
```

### Fuzzy Membership Functions

Linguistic variables defined using percentile-based cutpoints:

- **Low**: [min, 33rd percentile]
- **Medium**: [25th, 75th percentile] (overlapping transitions)
- **High**: [67th percentile, max]

### Monotonic Constraints

| Feature                 | Constraint | Economic Rationale                             |
| ----------------------- | ---------- | ---------------------------------------------- |
| `LIMIT_BAL`             | ‚Üë ‚Üí risk ‚Üì | Higher credit limit implies stronger borrowers |
| `AGE`                   | ‚Üë ‚Üí risk ‚Üì | Older applicants typically more stable         |
| `PAY_0`                 | ‚Üë ‚Üí risk ‚Üë | Recent delinquency signals distress            |
| `utilization`           | ‚Üë ‚Üí risk ‚Üë | High utilization indicates stress              |
| `repay_ratio1`          | ‚Üë ‚Üí risk ‚Üì | Higher repayment reduces risk                  |
| `delinquency_intensity` | ‚Üë ‚Üí risk ‚Üë | Historical delinquency compounds risk          |
| `paytrend`              | ‚Üë ‚Üí risk ‚Üì | Improving trend lowers risk                    |

## Datasets

## Research Impact

This framework demonstrates that regulatory compliance and model performance are not mutually exclusive. By combining fuzzy reasoning with monotonic gradient boosting, we achieve:

- **Structural Transparency**: Fuzzy rules provide human-readable explanations
- **Economic Consistency**: Monotonic constraints enforce domain knowledge
- **Competitive Performance**: Maintains accuracy comparable to black-box models
- **Regulatory Alignment**: Meets Basel II/III and IFRS-9 requirements
